## Tài liệu Dự án

- [Kế hoạch Khảo sát Hiện trạng (Baseline Data Architecture)](docs/project_plan.md) - Kế hoạch khảo sát chi tiết các nguồn dữ liệu, luồng thông tin, tích hợp và hoạt động dữ liệu hiện tại của Long Châu.

## Cấu trúc Thư mục

- `docs/`: Thư mục chứa tài liệu đặc tả kiến trúc.
  - `project_plan.md`: Kế hoạch khảo sát và phân rã công việc (WBS).
- `models/`: Thư mục chứa các tệp mô hình hóa kiến trúc (ArchiMate, DFD, v.v.).
